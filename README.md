# RS485-Bridge

RS485 USB Serial 를 통해 받은 데이터 처리 후 서버로 전송

## 사용 명령어 모음

```bash
# 서버 실행
python src/app/main.py
# DB -> csv
python src/tools/getDBdata.py
```

## 패키지 및 버전 관리

```bash
# venv
python3 -m  venv .venv
source .venv/bin/activate
deactivate
# package
pip freeze > requirements.txt
pip install -r requirements.txt
```

## 시리얼 설정
- /dev/ttyUSB0
- baudRate: 9600
- dataBits: 8
- stopBits: 1
- parity: 'none'

## PL2303 드라이버 설치시 유의점

- 2008 오류 발생시 대처법
    1. PL2303_Prolific_GPS_1013_20090319.exe 설치
    2. 장치관리자
    3. 포트
    4. 2008 포트 속성
    5. 드라이버 업데이트
    6. 내컴퓨터에서 드라이버 찾아보기
    7. 컴퓨터의 사용가능한 드라이버 목록에서 직접 선택
    8. 2008년 드라이버 선택
